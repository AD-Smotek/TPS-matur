# Typy pamětí a jejich použití

## 1) Co je počítačová paměť

Počítačová paměť je zařízení pro **uložení dat a instrukcí**, které procesor zpracovává.

Paměť se dělí podle **rychlosti, volatility a účelu**.

---

## 2) Hlavní typy pamětí

### 1) RAM (Random Access Memory)
- **Typ:** volatilní (po vypnutí se data ztratí)  
- **Použití:** běh programů a operačního systému  
- **Druhy:**
  - **DRAM** – běžná operační paměť, levnější, pomalejší  
  - **SRAM** – rychlejší, dražší, používá se v cache  
- **Funkce:** ukládá aktuálně používaná data a instrukce

---

### 2) ROM (Read-Only Memory)
- **Typ:** nevolatilní  
- **Použití:** ukládání základního softwaru (BIOS/UEFI)  
- Nelze běžně zapisovat, jen číst  
- Varianty:
  - PROM – programovatelná jednou  
  - EPROM – mazatelná UV světlem  
  - EEPROM – mazatelná elektrickým signálem  

---

### 3) Cache paměť
- **Typ:** velmi rychlá paměť uvnitř procesoru  
- **Účel:** zrychlení přístupu k často používaným datům  
- **Úrovně:**
  - L1 – nejrychlejší, přímo v jádře CPU
  - L2 – větší, pomalejší
  - L3 – sdílená mezi jádry

---

### 4) Flash paměť
- **Typ:** nevolatilní  
- **Použití:** USB disky, SSD, paměťové karty  
- Rychlá čtení, pomalejší zápis  
- Snadno přenosná

---

### 5) Virtual Memory (virtuální paměť)
- **Typ:** kombinace RAM a diskového prostoru  
- **Použití:** umožňuje spouštět více programů než kolik je fyzické RAM  
- **Princip:** část disku se používá jako dočasná RAM (page file / swap)

---

### 6) Registr v CPU
- **Typ:** extrémně rychlá paměť uvnitř procesoru  
- **Použití:** uchovává dočasné hodnoty pro výpočty  
- Velikost: jen několik bajtů na registr

---

### 7) ROM + RAM hybridy
- Například **BIOS/UEFI** využívá ROM pro start systému a RAM pro runtime nastavení.

---

## 3) Shrnutí

| Typ paměti | Volatilní | Použití |
|------------|-----------|---------|
| RAM        | Ano       | Běh programů a OS |
| ROM        | Ne        | Ukládání BIOS/firmware |
| Cache      | Ano       | Rychlý přístup k datům CPU |
| Flash      | Ne        | SSD, USB, paměťové karty |
| Virtuální  | Ano/Ne    | Rozšíření RAM pomocí disku |
| Registr    | Ano       | Dočasné výpočty CPU |

Paměť je klíčová pro **rychlost, stabilitu a efektivitu počítače**.  
Různé typy pamětí se doplňují a mají specifické použití podle rychlosti, ceny a volatility.
